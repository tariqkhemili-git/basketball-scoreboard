# Basketball Scoreboard

A simple, elegant basketball scoreboard web application with a retro digital display aesthetic.

## Features

- **Dual Score Tracking**: Separate score displays for HOME and GUEST teams
- **Multiple Increment Options**: Add 1, 2, or 3 points per button click
- **Visual Leader Indicator**: Leading team's score highlighted in yellow (#ffe600)
- **Auto-Reset**: Scores automatically reset to 0 when exceeding 99
- **New Game Function**: Quick reset button to start a fresh game
- **Retro Design**: Dark blue background with styled digital scoreboard displays

## Technologies Used

- HTML5
- CSS3 (Flexbox layout)
- Vanilla JavaScript
- Custom "Cursed Timer ULiL" font for authentic scoreboard look

## How to Use

1. Click the +1, +2, or +3 buttons under each team to increment their score
2. The leading team's score will turn yellow automatically
3. Click "new game" to reset both scores to 0

## Color Scheme

- Background: `#1b244a` (Navy Blue)
- Score Default: `#f94f6d` (Pink)
- Leader Score: `#ffe600` (Yellow)
- Buttons/Borders: `#9aabd8` (Light Blue)

Perfect for keeping score during pickup basketball games or as a learning project for web development basics!
